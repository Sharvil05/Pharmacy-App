# pharmacyapp

Online Pharmacy App built using Flutter & Firebase with wallet system, product browsing, orders, authentication, and clean UI.

# Features

* User Authentication (Login / Signup)
* Home page with medicine listing
* Category-based product filtering
* Medicine details page
* Wallet system (fake wallet with debit & credit)
* Order placement & order history
* Profile page with user details
* Splash Screen with auth check
* Clean & responsive UI

### Tech Stack
* Flutter (Dart)
* Firebase Authentication
* Cloud Firestore
* Shared Preferences
* Material UI

### How to Run
flutter pub get
flutter run
